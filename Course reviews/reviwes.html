ITNE352 Network Programming

Dr. Mohammed Abdulaziz ★★★★☆ (4/5) Fall 2023

Course Review

This course provides comprehensive coverage of network programming concepts including socket programming, TCP/IP protocols, and client-server architectures. The assignments were challenging but well-structured to build practical skills. Dr. Abdulaziz explains complex topics clearly and provides excellent support during office hours.

Course Highlights

  • Hands-on socket programming exercises
  • Detailed coverage of network protocols
  • Practical client-server project
  • Clear lecture materials

ITCS333 Internet Software Development

Mr. Abdulla Ebrahim ★★★★★ (5/5) Summer 2023

Course Review

One of the most practical courses in the curriculum. Mr. Ebrahim's approach to teaching web development is project-based, which helped me understand real-world applications immediately. We covered frontend technologies (HTML5, CSS3, JavaScript) and backend integration with databases.

Course Highlights

  • Complete web application development
  • Modern JavaScript frameworks

  • Helpful code reviews
